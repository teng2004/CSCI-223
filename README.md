This project is an autocomplete system which provide words suggestion for the user by the prefix they type in. The system is built upon Trie and DFS algorithm which is fairly efficient to find words that have the same prefix.
To run this program, we need to open the terminal and type "python autocomplete.py". Then the user can enter the prefix to see the suggestion list or type "exit" to leave.
In order to make the system work, we need autocomplete.py which design the data structure and algorithm. We also need a dictionary.txt to be the database, so the program can find words from the dictionary and generate a suggested word list.
The algorithm has main three steps. First, it will read every single words in the dictionary and then start navigating from the root. If there is not a node for the letter of a word, it would create a new node for it. Second, when the user type in the prefix, the algorithm starts from the root in the trie and traverse til the end. Once it reaches the end of the prefix, it starts doing DFS to find out all the subtrie and put them in a list.
The time complexity of the inserting is O(L). To insert a word, we traverse L nodes where L is the length of the word. The time complexity of Search Prefix is O(P) which P is the length of the prefix. Finally, the suggestion time complexity is O(N) where N is the total number of the children node of the prefix.
